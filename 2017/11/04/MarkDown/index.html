MarkDown教程

MarkDown简介

宗旨

 MarkDown是一种轻量级的标记语言，创始人约翰·格鲁伯(John Gruber)。它允许人们使用易读易写的纯文本格式编写文档，然后转换成有效的HTML文档或PDF文档。

 MarkDown的语法全是由一些符号组成，使人注意力的更多的关注在代码内容上，而不是文本格式上。

兼容HTML

不在MarkDown覆盖范围内的标签，都可以直接在文档里面用HTML撰写，不需要额外的标注是HTML或者是MarkDown，只要直接加标签就可以了。

MarkDown基本语法

标题

标题1：#标题1
标题2：##标题2
标题3：###标题3
标题4：####标题4
标题5：#####标题5
标题6：######标题6

结果

标题1

标题2

标题3

标题4

标题5
标题6

换行

在文字的末尾使用两个或两个以上的空格就好。

引用

行首使用”>+空格”表示引用段落，内部可以嵌套多个引用。

这是一个引用

内部嵌套

列表

无序列表使用*、+、-后面加上空格来表示。

语法示例：

* 列表1
+ 列表2
- 列表3

结果：

  • 列表1
  • 列表2
  • 列表3

有序列表使用数字+英文句号+空格表示，数字不对也可以自动换成正确的序号。

语法：

1.列表1
2.列表2
7.列表3

结果：

  1. 列表1
  2. 列表2
  3. 列表3

强调

MarkDown使用*或者_表示强调

语法:

单星号 = *斜体*
单下划线 = _斜体_
双星号 = **加粗**
双下划线 = _加粗_
删除线 = ~~删除~~
水平线 =
---
***
___

结果：

单星号 = 斜体
单下划线 = 斜体
双星号 = 加粗
双下划线 = 加粗
删除线 = 删除
水平线 =

链接

语法：
[链接文字](url)

示例：
[郭靖的博客](http://guojjing.github.io)

结果：
郭靖的博客

图片

语法：
![图片的替代文字](图片url)

示例：
![兔子](https://i.loli.net/2017/10/20/59e97f6b0d315.jpg)

结果：

# 转载
