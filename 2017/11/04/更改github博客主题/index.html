更改github博客主题

发表于 2017-11-04 | 分类于 教程

轻松更换博客的主题

在开始更换之前，要先选一个心仪的主题对不对？这里放一个官方主题的链接，点我

以下以CyanStyle主题为例。

注意，鼠标点击图片会进入预览，我们这里要点击蓝字，进入资源网址

然后下滑，下滑，没错，这里有教程！


如图，有教程的可以直接copy②的链接，然后下载资源文件，如果没有，就复制①的网址，然后按照 git clone 网址.git themes/主题名称 的格式下载资源文件。

1.在blog中右击，git clone把资源文件下载到本地

2.升级我个人认为都是最新的，然后这一步就省略了，你如果想更新的话，就在下载好的主题文件下，文件目录在blog/themes，右击 git pull

3.修改配置文件blog下的_config.yaml（这里约定一下叫站点配置文件），把theme改为主题名称

顺便配置一下页面信息

注意language一项必须是zh-Hans，中文

4.打开主题文件写下的_config.yaml（这里称为主题配置文件）

这里各个部分的含义，资源网址中都有，这里就不一一说了，重点讲一下favicon（页面标题的图标）

和music音乐

前排放几个要用到的网址

①图片转链接
②制作ico图标

打开②网址，把你喜欢的图片制作成ico格式，然后打开①网址图床生成链接，把链接修改在favicon处就可以了

上传音乐同理，需要把mp3文件转换为链接，这里我提供一个方法，打开qq邮箱

点击①文件中转站，然后点击②上传，把本地的mp3文件上传上去
然后点击③下载，

点击复制链接地址
把链接的mp3以后的都删掉，得到的这个网址就可以放到music中。
（我用的qq浏览器，效果可能不同）

所有的配置都完成后,在blog中打开git，输入 hexo clean ; hexo genarate 回车，一顿翻滚平静之后输入 hexo deploy

字幕滚完，新博客也就可以访问了。

每个主题的元素都不一样，配置方式也不一样，此教程仅供参考，不可照搬。

具体请按照资源网址的教程进行配置。
